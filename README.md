### Jak używać

1. Kliknij link powyżej — otwiera się w przeglądarce (Chrome / Edge / Firefox)
2. Wybierz **kategorię** (ETF, Krypto, Akcje, Surowce, Forex, Indeksy)
3. Wybierz **instrument** z listy
4. Kliknij **🤖 Analizuj** — agent generuje pełną analizę

---

### Funkcje

| Funkcja | Opis |
|---|---|
| 📊 Wykres świecowy | Interaktywny wykres z EMA 21/50/200, Bollinger Bands, S/R |
| 🤖 Analiza AI | Bias rynku, sygnał BUY/SHORT/CZEKAJ z poziomami |
| 📐 Fibonacci | Automatyczne zniesienia Fibonacci na wykresie |
| 📈 RSI + MACD + Vol | Sub-wykresy pod głównym wykresem |
| 🕯️ Formacje | 11 formacji świecowych z opisami po polsku |
| 🧮 Kalkulator | Kalkulator ryzyka i position sizing (PLN) |
| 🔴🟢 Linie S/R | Automatyczne wsparcia i opory dla każdego instrumentu |
| 🔴🟢 Linie trendu | Rosnące wsparcia i opadające opory |
| 📡 Dane live | Krypto: CoinGecko API · Reszta: Yahoo Finance (15 min delay) |
